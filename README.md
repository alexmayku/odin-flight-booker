# Flight Booker

This is a Rails project from The Odin Project.  
It focuses on building and handling forms, including nested parameters and multi-step user input.

Part of the Flight Booker project in the Rails course.

## What this does
- Lets users search for available flights
- Select a flight
- Enter passenger information via forms
- Create a booking with associated passengers

The main goal is learning Rails form helpers, params, and associations.

## Tech
- Ruby on Rails
- ERB templates
- ActiveRecord associations

## Key concepts practiced
- `form_with`
- Nested forms
- Strong parameters
- One-to-many associations
- Passing data between requests
